# Code of Conduct

This project aims to be welcoming, practical, and respectful.

## Expected behavior

- Be respectful and constructive.
- Focus feedback on ideas, code, and outcomes rather than people.
- Share context generously, especially when reviewing or disagreeing.
- Assume good intent, while still being clear and honest about problems.
- Help keep discussions accessible to people with different levels of experience.

## Unacceptable behavior

- Harassment, threats, or personal attacks
- Discrimination or exclusionary language
- Deliberate intimidation or sustained hostility
- Publishing private or sensitive information without consent
- Spam, trolling, or bad-faith disruption

## Enforcement

Project maintainers may edit, remove, or reject comments, issues, pull requests, or contributions
that violate this code of conduct.

## Reporting

If you experience or witness behavior that should be reported, contact the maintainers through a
private channel when possible. If no private channel is available yet, open a minimal public issue
without sensitive details and request a confidential follow-up path.
